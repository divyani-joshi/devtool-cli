# DevTool CLI

A developer productivity toolkit built with Node.js.

## Features

- Password Generator
- React Component Generator
- Git Status Command

## Installation

```bash
npm install
```

## Usage

```bash
node index.js password 16
node index.js component Home
node index.js git-status
```

## Technologies

- Home.js
- Commander.js
- Chalk
